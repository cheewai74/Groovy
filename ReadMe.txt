GRoovy Advanced Packaging Engine(Grape) 


docker run -it --rm groovy

grape list

To install artifacts before execution use “grape install” command
>>grape install [-hv] <group> <module> [<version>] [<classifier>]
>>grape uninstall [-hv] <group> <module> <version>
